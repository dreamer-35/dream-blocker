# dream-blocker

开源浏览器扩展集合，用于在常见网站上屏蔽垃圾内容与引流账号。

## 项目

| 目录 | 说明 |
|------|------|
| [x-yellow-blocker](./x-yellow-blocker/) | Chrome 扩展：X（Twitter）网页版黄推/引流自动隐藏与一键拉黑 |

## 快速开始

```bash
git clone https://github.com/dreamer-35/dream-blocker.git
cd dream-blocker/x-yellow-blocker
```

然后在 Chrome 打开 `chrome://extensions` → 开发者模式 → **加载已解压的扩展程序** → 选择 `x-yellow-blocker` 目录。详细说明见各子项目 README。
